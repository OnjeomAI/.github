## 프로젝트 소개

**온점(·)** 은 문장을 완성하는 마침표이자, 세상을 온전히 읽는 시작점입니다.

문해력이 저하되는 시대, 사용자의 학습 데이터를 분석하여 **어휘력 · 독해력 · 문해력**을 맞춤형으로 향상시키는 AI 기반 학습 앱입니다. 학생부터 성인까지, 모두를 위한 문해력 솔루션을 목표로 합니다.

## 핵심 기능

| 기능 | 설명 |
|:----:|------|
| 문해력 진단 | 초기 테스트로 개인별 어휘력·독해력·문해력 수준 정밀 분석 |
| 맞춤형 학습 | AI가 부족한 영역 중심으로 맞춤 콘텐츠 제공 |
| 적응형 난이도 | 오답 패턴 및 학습 이력 분석 → 자동 난이도 조절 |
| 학습 시각화 | 성장 추이를 수치와 그래프로 직관적 확인 |
